# MaterialDesignWeibo

Material Design风格的微博客户端，可以使用大部分和新浪微博相似的功能，但某些功能因为新浪接口限制不能使用。</br>
使用第3方网站解析了一些视频网站的源地址，直接可以用内置播放器播放，不用跳转到网页播放视频（在64位系统上不能使用）。</br>
视频链接解析为第3方提供，所以仅用于学习交流，不可用于商业用途，如有侵权告知删除。
